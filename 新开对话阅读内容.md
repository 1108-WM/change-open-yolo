## 2026-07-22 最新优先级：先修超点原子，再验证 SAM2

- 原始 ScanNet superpoint 对有效 GT 实例的边界覆盖为 100%，当前 f30 IBSp 为 86.2%；对 Mask3D 漏检实例为 100% 对 75.8%。下降发生在 f30 IBSp 的全 mesh 重分割，早于可靠性筛选、采样、SAM2 和 Details Matter。
- 高覆盖只是“真实边界没有被原子 superpoint 吞并”的上界，不等于 AP。未加 SAM2 的强基线是 `Mask3D + YOLO-World + SAM + BPR -> MVPDist -> superpoint 精炼 -> 评分/NMS`；历史 f30 主入口 AP 为 26.6595%，加现有 SAM2 候选为 26.6907%，仅增加 0.0312 个百分点，另一 f30 一致入口基本持平。因此 SAM2 当前没有稳定总体增益，不应继续叠加新模块或重跑轨迹。
- 下一步必须先将 IBSp 基础粒度与原始 ScanNet mesh-normal 分段对齐：以原始或结构等价的过分割作为原子底座，只根据可靠二维边界切分，不跨原始超点做早期合并；对象合并延后给多视角 SAM2、二维边界、MVPDist 语义共同决定，Mask3D 保持回退。完成无 GT 的结构对齐检查前，不重跑 SAM2。
- 对“先过分割、再合并”的判断：这是正确方向。欠分割后再拆分通常不可控；适度过分割可由多视角和语义证据保守合并。
