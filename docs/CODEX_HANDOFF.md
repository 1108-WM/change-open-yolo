# Codex Handoff

这是新会话或换设备时的快速入口，不是实验日志。

阅读顺序：

1. `新开对话阅读内容.md`
2. `CURRENT_EXPERIMENT_STATUS.md`
3. `资料/当前基线修改方向.md`
4. `资料/论文阅读记录.md`
5. `资料/技术问题与答复.md`

文档分工：

- `CURRENT_EXPERIMENT_STATUS.md`：唯一的当前状态、关键结果和下一步。
- `资料/当前基线修改方向.md`：稳定的论文路线与模块边界。
- `资料/论文阅读记录.md`：论文依据与可迁移思路。
- 本文件：仅说明阅读顺序和工作边界。

工作边界：

- 默认只做诊断和 export-only；除非用户明确要求，不接主融合、不运行最终 AP。
- 不得使用 ScanNet200 GT 作为推理输入。
- 三场景仅作烟测；通过后用随机 `even48` 做方向筛选，再用 `even96/odd96` 确认。
- 保护已有未提交改动；不要提交或整理 `output/`、数据、权重、缓存、PLY 或编译产物。
- Mask3D 与 YOLO-World 为第三方主体，创新优先放在二维证据、IBSp、跨视角实例形成和候选层。
- 当前最高优先级是修正 f30 IBSp 的基础粒度：原始 ScanNet superpoint 的 GT 边界覆盖为 100%，当前 f30 IBSp 为 86.2%，下降发生在 SAM2 前。先以原始或结构等价的过分割为原子底座，只追加可靠二维切分，再验证 SAM2；不要先重跑 SAM2 或叠加新模型。

开始前：运行 `git status --short`，阅读当前任务相关代码和上述状态文件。
